# 0. Pre-req
* AWS Account
* AWS Cli
* Git

# 1. Setup
* Create IAM Role
* Configure AWS Cli
* Clone Repository

# 2. Build
* Give yourself executable permissions to the build script

`chmod u+x build.sh`

* Run the build script

` ./build.sh`

# 3. Deploy
* Give yourself executable permissions to the deploy script
`chmod u+x deploy.sh`
* Run the deploy script
` ./deploy.sh`
